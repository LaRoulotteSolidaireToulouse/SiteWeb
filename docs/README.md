📚 Documentation
###### SiteWeb La Roulotte Solidaire Toulouse

Ce dossier docs/ contient l’ensemble des documents techniques, guides, notes et ressources destinés à accompagner le développement, la maintenance et l’évolution du site web La Roulotte Solidaire Toulouse

---

📁 Contenu du dossier

- Structure du projet, organisation des fichiers, logique des dossiers  
- Les fichiers présents peuvent varier selon l’avancement du projet.

---

🎯 Objectifs de la documentation

- Faciliter la compréhension du projet pour les contributeurs  
- Offrir une référence claire pour la maintenance et les mises à jour  
- Documenter les bonnes pratiques et les choix techniques  
- Centraliser les informations utiles pour le développement futur  
- Assurer une cohérence entre les différentes versions du site  

---

🛠️ Utilisation

- Lire architecture.md pour comprendre la structure globale  
- Lire workflow.md pour suivre les conventions du projet  
- Consulter les guides dans guides/ pour les tâches spécifiques  
- Ajouter toute nouvelle documentation dans un fichier clair et daté  
- Garder un style simple, professionnel et cohérent
