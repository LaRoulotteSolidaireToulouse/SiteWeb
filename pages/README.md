# 📄 Dossier pages/
###### SiteWeb La Roulotte Solidaire Toulouse

Le dossier pages/ contient l’ensemble des pages internes du site web La Roulotte Solidaire Toulouse.  
Ces fichiers HTML sont chargés via la navigation principale et respectent la structure, le style et les composants définis dans index.html.

---

📁 Contenu typique

- actions.html — Présentation des actions, projets et interventions de l’association  
- aider.html — Comment aider : dons, bénévolat, partenariats  
- infos.html — Informations pratiques, horaires, contacts, localisation  
- mentions-legales.html — Mentions légales obligatoires  
- contact.html (si présent) — Formulaire ou informations de contact  

Chaque page suit la même logique :  
- structure HTML minimaliste  
- classes cohérentes avec la feuille de style  
- sections claires et accessibles  
- contenu optimisé pour une lecture simple  

---

🎯 Objectifs du dossier

- Centraliser toutes les pages internes du site  
- Garantir une cohérence visuelle et structurelle  
- Faciliter la maintenance et l’ajout de nouvelles pages  
- Assurer une navigation fluide et intuitive  

---

🛠️ Bonnes pratiques

- Garder une structure HTML simple et propre  
- Réutiliser les classes et composants existants  
- Respecter la cohérence avec index.html  
- Nommer les fichiers en kebab-case  
- Ajouter uniquement des pages nécessaires à la navigation  

---

🔧 Intégration avec le site

Les pages sont appelées depuis :  
- la navigation principale  
- les liens internes  
- les boutons d’action  

Elles utilisent :  
- les styles du dossier css/  
- les images du dossier assets/  
- les scripts éventuels du dossier js/  

---

🤝 Contributions

Toute nouvelle page doit :  
- respecter la structure existante  
- être nommée clairement  
- être testée sur mobile et desktop  
- être ajoutée à la navigation si nécessaire
